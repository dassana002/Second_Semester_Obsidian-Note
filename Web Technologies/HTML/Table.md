==**Data rows (rows) සහ columns (columns)** වල organize කරලා show කරන structure එකක්.==

Example:
- student marks table
- timetable
- price list
- report table

HTML table represent කරන්නේ **2-dimensional data grid** එකක්.

```
<table>
    <caption> title </caption>

    <thead>
        <tr>
            <th>heading</th>
        </tr>
    </thead>

    <tbody>
        <tr>
            <td>data</td>
        </tr>
    </tbody>

    <tfoot>
        <tr>
            <td>summary</td>
        </tr>
    </tfoot>

</table>
```
