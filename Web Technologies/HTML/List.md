==**Items (Data / information)** එකක් structured (organized කරලා) display කරන්න use කරන HTML element එකක්.==

HTML list use කරන ප්‍රධාන හේතු:
- information organize කරන්න
- readability වැඩි කරන්න
- navigation menu create කරන්න
- steps show කරන්න
- grouped data show කරන්න

# Order List
==**Ordered List** එකක් කියන්නේ order (sequence) important වෙලා items numbers / letters වලින් show වෙන list එකක්.==
```
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
```

# Unorder List
==Order important නැති items show කරන list එක.==
```
<ul>
  <li>Apple</li>
  <li>Mango</li>
  <li>Orange</li>
</ul>
```

# Description List
==Term + Description pair show කරන list.==
```
<dl>
  <dt>HTML</dt>
  <dd>Hyper Text Markup Language</dd>

  <dt>CSS</dt>
  <dd>Style sheet language</dd>
</dl>
```

# Nested List
==List item එකක් ඇතුළත another list එකක් place කරන එක.==
```
<ul>
  <li>Fruits
    <ul>
      <li>Apple</li>
      <li>Mango</li>
    </ul>
  </li>

  <li>Vegetables</li>
</ul>
```
