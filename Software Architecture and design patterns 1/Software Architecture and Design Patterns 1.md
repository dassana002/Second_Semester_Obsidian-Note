# Software Architecture
==Software system එකේ **high-level structure (Over-view Idea)** එක.==
system එක කොටස් වලට බෙදලා ඒවා connect කරන plan එක.

System එකේ Major components කොහොම Organize වෙලා තියෙන්නේද,
